## About Project

A web application created with Symfony for Form'Aqui:

- [Form'Aqui](https://formaqui.fr//).
- [Symfony](https://symfony.com/).

### Installation

```bash
composer install
```

```bash
npm install
```

```bash
npm run watch
or
npm run build
```
